Olá, eu sou o Charles


  Sou um desenvolvedor autodidata que atualmente utiliza com mais frêquencia as linguagens Python e Javascript para resolver os problemas que aparecem no dia a dia. Além dessas já tive contato com outras linguagens como Haskell, Java, C++ e C.

  Dentro do mundo de banco de dados já tive que lidar com bancos relacionais e não relacionais, como MySql, MongoDB, Fire Base e SQLite, seja criando querys manualmente ou usando ORMs, em Python um ORM que utilizo é o SQLAchemy.

  Tenho interesse em matematica e otimização de tarefas, desde que iniciei na area de programação busco sempre desenvolver bons códigos, de forma a torna-los simples e melhores tanto para fins de otimização como de manutenção. Programar não é somente escrever comandos, é preciso entender o que se está criando, se planejar e buscar a melhor solução para o problema em questão. Devido a esse interesse tenho o habito de sempre me aprofundar em questões que envolvem designer patterns, clean code, analise de algoritmos e melhores praticas de programação.
